Download python 3, then download this. In the file directory bar when you open the folder type "cmd" and press enter. Use the py -3 -m pip install -r requirements.txt command to install all of the Python modules and packages listed in your requirements.txt file. This saves time and effort. Then send a message on discord, open inspect element, and go to the network tab. Open the package named "message" then copy the variable called authorization. Put the authorization token in the config file. Finally, right click and open the main.py file with python and have fun!
